## ChatXYZ 
ChatXYZ is a full-stack AI application built with Node.js + Express that enables users to interact with advanced AI models for chat, code generation, and image creation.

## 🚀 Features

* 💬 AI-powered chat system for real-time conversations
* 🎨 AI image generation from text prompts
* 🔐 Secure authentication with Google OAuth
* ⚡ High-performance backend built with Node.js and Express
* 🛡️ Rate limiting for API protection
* 🔒 Secure session management
* 🌐 RESTful API architecture
* 🌍 CORS-enabled for seamless frontend integration
* ⚙️ Environment-based configuration for flexible deployment
